# Founder Next Steps

## Immediate setup

1. Copy this starter pack into the root of the Elev8-OS repository.
2. Keep the existing plugin source in its own folder.
3. Commit the foundation documents.
4. Push the commit to GitHub.
5. Create the labels listed below.
6. Create the first issues.
7. Create a GitHub Project board.
