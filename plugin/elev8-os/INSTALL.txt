Elev8 OS 7.6.1 Installation

1. Back up the current Elev8 OS plugin and database.
2. Install the complete elev8-os-7.6.1-dashboard-coach-polish.zip through WordPress Plugins > Add New > Upload Plugin.
3. Replace the existing plugin when WordPress prompts you.
4. Activate Elev8 OS if needed.
5. Open the Artist Dashboard and verify the order: navigation, welcome, business snapshot, Up Next, Today's Focus.
6. Confirm recommendation buttons and existing artwork, QR, cart, checkout, and classes still work.
7. Test on Local before installing the exact same ZIP on Live.
