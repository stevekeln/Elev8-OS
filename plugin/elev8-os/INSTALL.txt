Elev8 OS 7.2.0 Installation

1. Back up the current site and retain the last stable Elev8 OS ZIP.
2. Install this complete plugin ZIP on Local.
3. Confirm the Artist Portal, My Artwork, existing classes, and public artist pages still load.
4. Edit an existing available artwork item and save it. Confirm WooCommerce connected appears.
5. Open the public artist page while logged out. Confirm the item is clickable and Add to Cart works.
6. Run a test checkout using a safe test payment method. Confirm the item becomes Reserved/Sold and cannot be purchased twice.
7. Install the exact tested ZIP on Live.

The database upgrade preserves existing artwork and adds inventory fields. Existing public artwork is automatically synchronized to WooCommerce when WooCommerce is active and the item has a price.
