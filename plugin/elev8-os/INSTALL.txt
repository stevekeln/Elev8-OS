Elev8 OS 9.1.1 Installation

1. Back up WordPress and the database.
2. Install elev8-os-9.1.1-artist-marketing-center.zip on Local.
3. Confirm the installed plugin version is 9.1.1.
4. Open Elev8 OS > Portal Setup and confirm Marketing Center is Ready.
5. Open Elev8 OS > Growth Settings and configure the Art Walk Assistant.
6. Sign in as a mapped artist and test Marketing > Send Test to Me.
7. Confirm SMTP delivery and unsubscribe behavior before sending a live student campaign.
8. Install the exact same tested ZIP on Live.
