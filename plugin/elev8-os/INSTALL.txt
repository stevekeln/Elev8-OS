Elev8 OS 7.2.2 Installation

1. Back up the current site and retain the last stable Elev8 OS ZIP.
2. Install this complete plugin ZIP on Local.
3. Open a public artist page while logged out.
4. Click Add to cart on an available artwork item.
5. Confirm the item is added and the WooCommerce Cart page opens immediately.
6. Return to the artist page and confirm View cart and Checkout are visible.
7. Complete a test order and confirm the inventory and artist email workflow still work.
8. Install the exact tested ZIP on Live.

No database migration is required. Existing artwork, WooCommerce products, carts, orders, and artist mappings are preserved.
