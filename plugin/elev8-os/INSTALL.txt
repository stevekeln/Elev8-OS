Elev8 OS 10.2.4 Installation

1. Replace the changed files in your local Git repository.
2. Commit to develop and push to GitHub.
3. Run Build Elev8 OS Release.bat.
4. Install the generated ZIP into Local.
5. Confirm Marketing and Content Studio each show the Print & QR card.
6. Confirm Open My Print Center reaches the artist Print Center.
7. Confirm Manage My Artwork reaches the artist artwork page.
8. Test on desktop and mobile before installing the exact same ZIP on Live.

Database changes: none.
