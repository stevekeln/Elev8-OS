Elev8 OS 7.3.0 Installation

1. Back up the current site and keep the known-good 7.2.2 ZIP.
2. Install this complete ZIP on Local and replace the existing Elev8 OS plugin.
3. Confirm artist pages, My Artwork, cart, checkout, and one QR code.
4. Open Settings > Permalinks and click Save only if /artwork/ links return 404. Activation normally flushes rewrite rules automatically.
5. Install the exact tested ZIP on Live.

No existing asset, artist, WooCommerce, order, or inventory data is removed.
