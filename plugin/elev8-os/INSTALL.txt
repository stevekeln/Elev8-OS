Elev8 OS 10.2.3 Installation

1. Replace the listed changed files in the local Git repository on develop.
2. Commit and push to GitHub.
3. Run Build Elev8 OS Release.bat.
4. Install the BAT-generated ZIP into Local.
5. Test the Artist Dashboard, website navigation, Content Studio, Marketing, Growth Center, Business GPS, Artist Success, Print Center, public artist profiles, and QR output.
6. Install the exact same generated ZIP on Live only after Local passes.

Do not install the changed-files package directly into WordPress.
Database changes: none.
