Elev8 OS 7.8.0 Installation

1. Install this ZIP on Local WordPress first.
2. Activate or replace the existing Elev8 OS plugin.
3. Open Elev8 OS > Print & Identity.
4. Upload the standard paisley background and Elev8 Arts logo.
5. Open a public artist profile and test Print Artist Card and Print QR Code.
6. Test both single-card and two-per-sheet printing.
7. Confirm View Cart and Checkout display as separate buttons.
8. Install this exact ZIP on Live only after Local testing passes.
