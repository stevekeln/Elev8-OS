Elev8 OS 8.0.0 — Gallery Print Center

1. Install this ZIP on Local first.
2. Open Elev8 OS > Print Center.
3. Select an artist or artwork and preview the print format.
4. Confirm the QR opens the correct tracked public page.
5. Print or choose Save as PDF in the browser print dialog.
6. After Local testing, install this exact ZIP on Live.

Only WordPress administrators with manage_options can print. Artists continue maintaining their own approved profile and artwork information.
