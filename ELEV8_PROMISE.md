# The Elev8 Promise

We do not build software merely to add features.

Every meaningful feature in Elev8 OS must do at least one of the following:

- Save time
- Increase revenue
- Reduce mistakes
- Improve the artist experience
- Improve the customer experience
- Improve decision-making

If a proposed feature does none of these things, it does not belong.

## North Star

> Did this make Elev8 Arts easier to run?

## Artist Principle

> Artists should be able to run their Elev8 work from Elev8 OS without needing to log into Amelia.
