# Support

Elev8 OS is currently an internal project for Elev8 Arts.

## Internal support process

When something breaks:

1. Record what page or action caused the problem.
2. Record the plugin version.
3. Take a screenshot.
4. Copy the exact error message.
5. Record whether the issue happened as administrator, artist, customer, or visitor.
6. Add a bug report in GitHub.
7. Do not repeatedly retry financial or destructive actions until the problem is understood.

## Emergency production issue

If an update causes a critical error:

1. Deactivate the plugin through WordPress or the hosting file manager.
2. Restore the previous known-good plugin version.
3. Check the WordPress error log.
4. Restore the database backup only when necessary.
5. Document the incident before attempting another release.
