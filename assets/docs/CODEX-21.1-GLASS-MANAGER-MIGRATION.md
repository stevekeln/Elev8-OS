# Codex Assignment — 21.1 Glass Manager Workspace Migration

## Objective
Convert the existing Glass Manager Operational Home into registered workspace widgets rendered by the Responsive Grid Service.

## Constraints
- Do not change production, QC, pay, assignment, or permission business logic.
- Reuse current services as widget data sources.
- Use the centralized Access Service.
- Preserve all existing URLs and actions.
- One column below 600px; no horizontal overflow at 320px width.
- Report a Problem must remain visible.
- Do not add theme-specific selectors.

## Deliverables
- Registered Studio workspace widgets.
- Glass Manager template assembled from the shared grid.
- Mobile, tablet, and desktop acceptance tests.
- Before/after screenshots.
- Exact changed-file list and rollback notes.
