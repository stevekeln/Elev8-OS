# Roadmap

This roadmap is organized by milestones rather than by random feature accumulation.

## Milestone 1 — Foundation

**Goal:** Make the current plugin reliable, understandable, secure, and maintainable.

- [ ] Audit the existing plugin source
- [ ] Establish clean plugin folder structure
- [ ] Define database versioning
- [ ] Define roles and permissions
- [ ] Stabilize artist partnership rules
- [ ] Stabilize artist profile saving
- [ ] Stabilize public artist pages
- [ ] Stabilize booking destination buttons
- [ ] Stabilize contact, payment, and social links
- [ ] Stabilize referral tracking and QR codes
- [ ] Stabilize artist portal privacy
- [ ] Add systematic error logging
- [ ] Add migration and rollback strategy
- [ ] Create release checklist
- [ ] Create staging test process

## Definition of priority

- **Critical:** Broken, unsafe, financially incorrect, or blocks core use
- **High:** Significant time, revenue, artist, or customer impact
- **Medium:** Useful improvement with manageable impact
- **Low:** Polish or convenience
- **Someday:** Worth preserving but not ready
