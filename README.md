# Elev8 OS

**An operating system for creative studios.**

Elev8 OS is a WordPress platform being built first for Elev8 Arts in Colorado Springs. It connects artists, classes, bookings, partnerships, payouts, public artist pages, referrals, documents, reporting, and future business intelligence in one place.

## Mission

> Help creative businesses spend less time managing and more time creating.

## The Elev8 Promise

We do not build software merely to add features. We build software to solve real problems.

Every meaningful feature must do at least one of the following:

- Save time
- Increase revenue
- Reduce mistakes
- Improve the artist experience
- Improve the customer experience
- Help owners make better decisions

Our north-star question is:

> Did this make Elev8 Arts easier to run?

## Current foundation

The existing plugin foundation includes work around:

- Artist partnership rules
- Artist payout calculations
- Private artist portal
- Public artist profile pages
- Booking destination buttons
- Social, contact, and payment links
- Referral links and QR codes
- Tax-document fields
- Amelia and WooCommerce integration
- Development roadmap and release planning

## Founders Edition direction

The next major milestone will establish a dependable architecture for:

- CEO dashboard
- Customer CRM
- Native waitlist engine
- Class profitability
- Class simulator
- Opportunity board
- Problem library
- Studio setup wizard
- Elev8 Brain recommendations
- Inventory and equipment management
- Marketing insights
- Multi-location support

## Project status

Elev8 OS is currently an internal, private project under active development for Elev8 Arts. The repository should remain private until the founders intentionally decide otherwise.
