ELEV8 OS 10.10.2 REPOSITORY RECOVERY

This package was reconstructed from the Elev8 OS 10.10.2 plugin downloaded from the live WordPress file manager.

1. Keep GitHub Desktop on the develop branch.
2. Copy everything INSIDE this folder into C:\GitHub\Elev8-OS.
3. Choose Replace files when Windows asks.
4. Do not copy the enclosing recovery folder itself into the repository.
5. Run Build Elev8 OS Release.bat.
6. Confirm the builder finds only this plugin entry file:
   C:\GitHub\Elev8-OS\plugin\elev8-os\elev8-os.php
7. Install the generated ZIP into Local and test.
8. Commit the restored source to develop and push to GitHub.

Expected source version: 10.10.2
