ELEV8 OS 10.9.0 INSTALLATION

1. Replace the changed files in your local Git repository.
2. Commit to develop and push to GitHub.
3. Run Build Elev8 OS Release.bat.
4. Install the generated ZIP into Local.
5. Test with separate manager, retail employee, artist/teacher, and Open Mic DJ accounts.
6. Create a few CRM flyer locations and one outreach campaign.
7. Mark a stop delivered and confirm its last-delivery date updates.
8. After Local passes, install the exact same generated ZIP into Live.

Role note: Elev8 Teacher, Elev8 Open Mic DJ, and Elev8 Retail Employee roles are created automatically when the updated plugin loads.
