Elev8 OS v10.4.11 Installation

1. In Local WordPress, open Plugins > Add New > Upload Plugin.
2. Select the v10.4.11 installable ZIP.
3. Choose Replace current with uploaded when WordPress detects the existing plugin.
4. Keep Elev8 OS activated.
5. Open Elev8 OS > CEO Dashboard.
6. Confirm the new System Information section appears.
7. Confirm the public homepage still opens while logged in as an artist.
8. Confirm normal artist login still redirects to the Artist Dashboard.

Install this ZIP in Local first. After testing, install the exact same ZIP on Live.
