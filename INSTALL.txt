ELEV8 OS STANDARD RELEASE BUILDER v1.2

Replace these items in C:\GitHub\Elev8-OS:

1. Build Elev8 OS Release.bat
2. BUILD_VERSION
3. tools\build-release.ps1

You may replace the entire tools folder from this package.

Then double-click Build Elev8 OS Release.bat.
Install ONLY the newly generated main elev8-os-...zip from the releases folder into Local WordPress.
Do not install this builder ZIP into WordPress.
